Проверка зарезервированных ключевых слов

Для начала проверим

select distinct catdesc from pg_get_keywords();

catdesc

не зарезервировано

не зарезервировано \(но не может быть именем типа или функции\)

зарезервировано

зарезервировано \(но может быть именем типа или функции\)

Для проверки зарезервированных слов

select * from pg_get_keywords() where catdesc = 'зарезервировано';

и для не зарезервированных

select * from pg_get_keywords() where catdesc = 'не зарезервировано';

А теперь создадим таблицу

create table ALTER(ID int);

Запрет и разрешение соединения с базой

Запрет новых соединений с базой

update pg_database set datallowconn = 'false' where datname = '<имя базы>';

Разрешение новых соединений с базой

update pg_database set datallowconn = 'true' where datname = '<имя базы>';

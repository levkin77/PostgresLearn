Даты и интервалы

Для понимания как работать с интервалами выполните следующие запросы и проверьте ваши ожидания и результаты.

SELECT ('2017-01-31'::date + '1 month'::interval); SELECT ('2017-01-31'::date + '1 month'::interval) + '5 month'::interval; SELECT '2017-01-31'::date + ('1 month'::interval + '5 month'::interval); SELECT ('2017-01-31'::date + '5 month'::interval) + '1 month'::interval; SELECT ('2017-01-31'::date + '6 month'::interval);

Результаты

2017-02-28 00:00:00.000000 2017-07-28 00:00:00.000000 2017-07-31 00:00:00.000000 2017-07-30 00:00:00.000000 2017-07-31 00:00:00.000000

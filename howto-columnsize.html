Объем колонки в таблице

Описание таблицы для psql, где testidx - наименование таблицы

\d testidx

Для получения всех колонок таблицы:

select v.ordinal_position, v.column_name, v.data_type from information_schema.columns v where v.table_schema='public' and v.table_name='testidx' order by ordinal_position;

результат:

ordinal_position

column_name

data_type

1

id

bigint

2

fk_id

bigint

3

state

text

4

amount

numeric

5

item

text

6

created_at

timestamp with time zone

и данные о размере на диске для колонки:

select pg_size_pretty(sum(pg_column_size(id))) as total_size from testidx; select pg_size_pretty(sum(pg_column_size(state))) as total_size from testidx;

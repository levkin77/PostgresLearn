Установка наименование приложения для сессии

Для установки наименования приложения для сессии:

SET application_name = 'test_activity';

Это позволяет быстро найти нужное в для просмотра активности или правильно группировать приложения

select * from pg_stat_activity;
select * from pg_stat_activity where application_name='test_activity';

Как выполнять запросы с автоповторением

Для повторения выполнения запросов подключаемся к psql.
Например, для имеем запрос

select count(*) from public.testidx;

для выполнения запроса каждые три секунды:

\watch 3

Как получить текущее наиенование базы

select current_database();

или для psql

\conninfo

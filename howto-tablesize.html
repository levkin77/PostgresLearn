Размер таблицы или индекса

Для получения информации о таблице в psql

\d test

Размер таблицы на диске

select pg_size_pretty(pg_relation_size('testidx'));

Размер всех индексов для таблицы

select pg_size_pretty(pg_indexes_size('testidx'));

Размер одного индекса

select pg_size_pretty(pg_total_relation_size('testidx_pk'));

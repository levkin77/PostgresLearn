Секционирование таблиц по UUID

Как сделать patrition table по UUID

CREATE TABLE t1 (hash uuid, data text) PARTITION BY range(hash); CREATE TABLE t1p0 PARTITION OF t1 FOR VALUES FROM ('00000000-0000-0000-0000-000000000000') TO ('10000000-0000-0000-0000-000000000000'); CREATE TABLE t1p1 PARTITION OF t1 FOR VALUES FROM ('10000000-0000-0000-0000-000000000000') TO ('20000000-0000-0000-0000-000000000000'); CREATE TABLE t1p2 PARTITION OF t1 FOR VALUES FROM ('20000000-0000-0000-0000-000000000000') TO ('30000000-0000-0000-0000-000000000000'); CREATE TABLE t1p3 PARTITION OF t1 FOR VALUES FROM ('30000000-0000-0000-0000-000000000000') TO ('40000000-0000-0000-0000-000000000000'); CREATE TABLE t1p4 PARTITION OF t1 FOR VALUES FROM ('40000000-0000-0000-0000-000000000000') TO ('50000000-0000-0000-0000-000000000000'); CREATE TABLE t1p5 PARTITION OF t1 FOR VALUES FROM ('50000000-0000-0000-0000-000000000000') TO ('60000000-0000-0000-0000-000000000000'); CREATE TABLE t1p6 PARTITION OF t1 FOR VALUES FROM ('60000000-0000-0000-0000-000000000000') TO ('70000000-0000-0000-0000-000000000000'); CREATE TABLE t1p7 PARTITION OF t1 FOR VALUES FROM ('70000000-0000-0000-0000-000000000000') TO ('80000000-0000-0000-0000-000000000000'); CREATE TABLE t1p8 PARTITION OF t1 FOR VALUES FROM ('80000000-0000-0000-0000-000000000000') TO ('90000000-0000-0000-0000-000000000000'); CREATE TABLE t1p9 PARTITION OF t1 FOR VALUES FROM ('90000000-0000-0000-0000-000000000000') TO ('A0000000-0000-0000-0000-000000000000'); CREATE TABLE t1pa PARTITION OF t1 FOR VALUES FROM ('A0000000-0000-0000-0000-000000000000') TO ('B0000000-0000-0000-0000-000000000000'); CREATE TABLE t1pb PARTITION OF t1 FOR VALUES FROM ('B0000000-0000-0000-0000-000000000000') TO ('C0000000-0000-0000-0000-000000000000'); CREATE TABLE t1pc PARTITION OF t1 FOR VALUES FROM ('C0000000-0000-0000-0000-000000000000') TO ('D0000000-0000-0000-0000-000000000000'); CREATE TABLE t1pd PARTITION OF t1 FOR VALUES FROM ('D0000000-0000-0000-0000-000000000000') TO ('E0000000-0000-0000-0000-000000000000'); CREATE TABLE t1pe PARTITION OF t1 FOR VALUES FROM ('E0000000-0000-0000-0000-000000000000') TO ('F0000000-0000-0000-0000-000000000000'); CREATE TABLE t1pf PARTITION OF t1 FOR VALUES FROM ('F0000000-0000-0000-0000-000000000000') TO (MAXVALUE);

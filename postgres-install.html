Установка PostgreSql

Куда лучше установить PostgreSql

  • Windows - для простого изучения и базовых возможностей, для разработки

  • Docker для windows - для разработки, тестирования. Быстро поставили, быстро удалили.

  • Установка в отдельную виртуальную машину Linux - наиболее удобный вариант для разработчиков и продвинутого изучения

  • При использовании Windows - присмотритесь к wsl2, в целях разработки и управления серверами является неплохой альтернативой

  • Для использования на уровне предприятия - установка только в выделенный nix сервер

Рано и поздно вам понадобятся дополнительные расширения, большинство из которых будет представлено как исходный код или готовые сборки для nix систем.

Большое количество доступных bash скриптов - также nix системы.

Средства мониторинга производительности - в основном nix системы.

Клиентcкие утилиты для разработчиков на sql - как windows, так и nix.

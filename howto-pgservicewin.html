Служба postgres на windows и перенос каталога данных

Раздел реестра HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\postgresql-16
где postgresql-16 соответствует 16 версии,
для 15 - соответственно будет postgresql-15
для 14 - соответственно будет postgresql-14

параметры запуска службы в разделе ImagePath:

"D:\Program Files\PostgreSQL\16\bin\pg_ctl.exe" runservice -N "postgresql-16" -D "C:\Program Files\PostgreSQL\16\data" -w

где собственно указан каталог данных

-D "C:\Program Files\PostgreSQL\16\data

Порядок действий:

  1. Останавливаем службу

  2. Переносим каталог на необходимый диск. Например, текущий каталог по умолчанию был на диске C и мы перенесем его на диск D

  3. Соответственно проверяем что разрешения для папки соответствуют разрешениям исходного каталога.

  4. Меняем значение в реестре на правильный путь и стартуем службу.

Postgres на windows и bat файлы

После переноса каталога данных рекомендуется подправить bat файлы для старта/остановки службы для корректной работы указав правильный путь к каталогу данных, например для reload.bat

echo off "D:\Program Files\PostgreSQL\16\bin\pg_ctl.exe" reload -D "D:\PGDATABASE" pause

Файлы для корректировки: reload.bat, restart.bat, stop.bat

База только для чтения

После установки значений требуется рестарт сервера! Это относится как к одной базе, так и всех баз!

Переключение в режим только-чтение для базы

alter database mydatabase set default_transaction_read_only=on;

Отключение режима только-чтение

alter database mydatabase set default_transaction_read_only=off;

Для перевода всех баз в режим только-чтение

alter system set default_transaction_read_only=on;

Для отключения

alter system set default_transaction_read_only=off;

для проверки

show default_transaction_read_only;

Для пользователя

alter user vao set default_transaction_read_only = on;

Для сесии

SET TRANSACTION READ ONLY

После установки значений требуется рестарт сервера! Это относится как к одной базе, так и всех баз!

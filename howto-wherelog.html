Где лог файлы

По умолчанию лог находится в каталоге данных, в папке log
Наиболее часто логи по умолчанию расположены
/var/log/postgresql
/var/lib/pgsql/data/pg_log/

select pg_current_logfile();

Результат

log/postgresql-2024-04-02_000000.log

Для проверки в какой папке находятся логи

show log_directory;

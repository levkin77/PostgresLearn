Установка в Rocky Linux

Обновляем систему

Начните с обеспечения актуальности вашей системы Rocky Linux. Откройте терминал и запустите следующие команды:

sudo dnf update

Установка PostgreSQL

Для установки PostgreSQL на Rocky Linux используйте следующую команду:

sudo dnf install postgresql-server postgresql-contrib

Эта команда установит как сервер PostgreSQL, так и дополнительные пакеты вкладов, предоставляющие полезные расширения и утилиты.

Инициализировать базу данных PostgreSQL

После установки инициализируйте кластер базы данных PostgreSQL, запустив:

sudo postgresql-setup --initdb

Эта команда создаст необходимую структуру каталогов и файлы конфигурации для PostgreSQL.

Запустите и включите PostgreSQL

Чтобы запустить службу PostgreSQL и включить ее автоматический запуск при загрузке, используйте эти команды:

sudo systemctl start postgresql sudo systemctl enable postgresql

Создание пользователя и базы данных PostgreSQL

Создадим нового пользователя и базу данных PostgreSQL. Замените your_user и your_database предпочитаемыми значениями:

sudo -u postgres createuser your_user sudo -u postgres createdb -O your_user your_database

Проверяем доступ к инструменту командной строки PostgreSQL, psql, с помощью следующей команды:

sudo -u postgres psql -d your_database

Управление службой PostgreSQL

Для управления службой PostgreSQL в Rocky Linux можно использовать следующие команды:

  • Старт службы:

sudo systemctl start postgresql

  • Остановка службы:

sudo systemctl stop postgresql

  • Перезапуск:

sudo systemctl restart postgresql

  • Проверка статуса:

sudo systemctl status postgresql
